# 第二周到当前版本的整体评估

评估基线为《第二周工作笔记》18页内容。第二周已经完成论文原理理解和第一版研究原型，
当前版本则从“四类材料流程验证”推进到了“73标签、有限生长、动态重网格和电磁属性连续”的
多组织研究型前处理工具。以下不把概念调研与可运行功能混为一谈。

## 第二周已经具备的能力

1. 理解论文采用FEM、超弹性和乘法分解 `F = F_el F_in`，SAT通过各向同性
   `growth_lambda` 增减，而不是直接缩放皮肤。
2. 能把MAT体素标签转换为共享节点四面体网格，并简化为 BONE/SOFT/SAT/SKIN 四类。
3. 使用可压缩Neo-Hookean材料、固定骨节点和增量Newton法求静力平衡。
4. 输出位移、`growth_lambda`、`J_total`、`J_elastic`、VTU和JSON。
5. 将粗FEM位移通过四面体重心坐标映射到独立高分辨率表面。
6. 已认识到四类材料、完全各向同性、无接触、无局部脂肪分布和无近不可压缩混合单元的不足。

## 本轮新增功能的验证结果

- 单元与集成测试：完整结果见测试命令输出。
- 两阶段合成模型减脂：四面体 `108 -> 100`，SAT单元 `76 -> 68`，
  累计SAT几何体积比 `0.980786`，标签集合保持。
- 真实73标签粗网格独立塌缩：31次塌缩，四面体 `10644 -> 10552`，
  最大逐标签几何体积漂移 `1.414e-5`，参考材料体积守恒误差 `1.225e-16`，无翻转。
- 真实模型完整减脂链路：目标生长比 `0.995`，实际几何比 `0.996969`，
  SAT单元 `1782 -> 1736`，质量变化 `-0.08780 kg`，标签保持且无负体积单元。
- 弹性历史行列式与求解结果的一致性最大相对误差 `6.653e-16`。

粗stride40模型计算出的总体重约115 kg不能当作真实受试者体重；该数值首先受到原始体素标签、
粗采样体积误差和材料密度映射影响。这里验证的是守恒关系和流程，不是人体测量准确性。

## 仍需优化的工作

### P0：决定能否形成可信论文结果

1. 将纯位移P1四面体迁移到FEniCSx/PETSc混合 `u-p` 或稳定近不可压缩单元，支持百万级网格。
2. 把目标SAT几何体积外层校正与每阶段重网格统一起来，直接控制最终80%/100%/120%结果。
3. 用人体实验或可靠文献标定主要组织的非线性、粘弹性和纤维参数，并做敏感性/不确定性分析。
4. 用影像或人体测量数据验证腰围、SAT厚度、器官位移和外轮廓，而不只验证数值守恒。

### P1：提升多组织力学可信度

1. 重网格后自动重建接触，并传递接触活跃集、摩擦或滑移历史。
2. 把当前塌缩的局部加权历史投影升级为积分点到新网格的保守L2投影，并量化应力误差。
3. 增加edge flip、顶点重定位、目标尺寸场和边界重投影，进一步改善最差单元质量。
4. 将血液、CSF、尿液、胆汁和空气占位固体替换为压力腔、流体、流固耦合或多孔介质模型。
5. 加入腹部优先、四肢差异、性别和BMI相关的空间SAT生长场，而不是所有SAT同倍率。

### P2：面向电磁仿真的交付

1. 对重网格后每个标签做闭合性、连通性、最小厚度和微小孤岛检查。
2. 建立目标电磁频率下的IT'IS参数版本锁定和可追溯材料清单。
3. 增加Sim4Life可接受的多区域网格/表面导出与自动材料赋值脚本。
4. 做形变前后电磁网格收敛测试，区分形状变化、材料变化和离散误差对场结果的贡献。

## 结论

第二周的成果是“理解并跑通论文核心思路”；当前成果已经成为“保持73组织语义、可增减SAT质量、
可动态改变SAT网格、可传递弹性历史并为电磁赋值保留材料字段”的研究原型。下一阶段工作的重点
不再是继续堆命令，而是大规模稳定求解、材料标定、接触历史和真实数据验证。
